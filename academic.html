Electronics: Embedded Programming

Suitable for students with basic programming experience looking to get into embedded programming. The course covers programming in C on the ATmega2560 based on MikroElektronika's curriculum. You have the option to either use or not use Arduino's library based on your comfort level. A board will be provided during tutoring hours, but if you want to practice at home, you will need to purchase your own. Note that we are following the PIC book, but we are not using a PIC.

Locations available: Mississauga, Markham
Electronics: Circuits (PCB schematic, layout)

Suitable for students with basic programming experience looking to get into basic circuits. The course covers everything you need to know to make your own PCB. This ranges from schematic capture's circuit design to board layout electrical characteristics. The program that will be used is Eagle CAD and the board house will be Seeed Studio.

Locations available: Mississauga
Electronics: FPGA (Verilog, VHDL)

Suitable for students with basic programming experience looking to get into hardware design. The course covers everything you need to know to code an edge detector DSP block on an FPGA.

Locations available: Mississauga
